# 네이버 서치 어드바이저 등록 가이드

Vercel 배포 후 네이버 검색에 사이트를 등록하는 방법입니다.

---

## 📋 사전 준비

✅ Vercel 배포 완료
✅ 도메인 주소 확보 (예: `https://pinpoint.vercel.app`)

---

## 🚀 등록 절차

### 1단계: 네이버 서치 어드바이저 접속

```
https://searchadvisor.naver.com/
```

네이버 아이디로 로그인하세요.

---

### 2단계: 사이트 등록

1. **"웹마스터 도구"** 클릭
2. **"사이트 등록"** 버튼 클릭
3. 배포된 사이트 URL 입력
   ```
   https://pinpoint.vercel.app
   ```
4. **"확인"** 클릭

---

### 3단계: 소유권 확인

#### 방법 1: HTML 태그 (추천) ✅

1. 네이버에서 제공하는 메타 태그 복사
   ```html
   <meta name="naver-site-verification" content="발급받은코드" />
   ```

2. `index.html` 파일 수정
   ```html
   <head>
     ...
     <meta name="naver-site-verification" content="여기에_발급받은_코드_붙여넣기" />
     ...
   </head>
   ```

3. Git에 커밋 & 푸시
   ```bash
   git add index.html
   git commit -m "Add Naver verification tag"
   git push origin main
   ```

4. Vercel에서 자동 재배포 (1~2분 소요)

5. 네이버 서치 어드바이저에서 **"소유권 확인"** 클릭

---

#### 방법 2: HTML 파일 (대안)

1. 네이버에서 제공하는 HTML 파일 다운로드
   ```
   naver1234567890abcdef.html
   ```

2. `public/` 폴더에 파일 복사
   ```
   public/naver1234567890abcdef.html
   ```

3. Git에 커밋 & 푸시

4. 소유권 확인

---

### 4단계: 사이트맵 제출

1. **"요청" 탭** 클릭
2. **"사이트맵 제출"** 클릭
3. 사이트맵 URL 입력
   ```
   https://pinpoint.vercel.app/sitemap.xml
   ```
4. **"확인"** 클릭

---

### 5단계: RSS 등록 (선택사항)

블로그나 뉴스가 있다면 RSS 피드도 등록할 수 있습니다.

---

### 6단계: 로봇룰 검증

1. **"검증" 탭** 클릭
2. **"robots.txt"** 클릭
3. 자동으로 검증됨
   ```
   https://pinpoint.vercel.app/robots.txt
   ```

---

## ⏱️ 색인 소요 시간

| 단계 | 소요 시간 |
|------|-----------|
| 소유권 확인 | 즉시 |
| 사이트맵 제출 | 즉시 |
| 첫 크롤링 | 1~3일 |
| 검색 결과 노출 | 3~7일 |

**예상: 일주일 내 검색 결과 노출** ✨

---

## 📊 모니터링

### 수집 현황 확인

1. **"검증" 탭** → **"수집 현황"**
2. 크롤링 통계 확인
   - 수집된 페이지 수
   - 마지막 수집 일시
   - 오류 페이지

### 검색 반영 확인

```
site:pinpoint.vercel.app
```

네이버 검색창에 위 명령어를 입력하여 색인 상태 확인

## 🎯 최적화 팁

### 1. 페이지 제목 최적화
```html

```

### 2. 메타 설명 추가
```html
<meta name="description" content="54개 주요 대학의 정시 합격 확률을 AI로 분석합니다." />
```

### 3. 키워드 최적화
```
주요 키워드:
- 대학 합격 예측
- 정시 합격률
- 수능 성적 분석
- 입시 컨설팅
```

### 4. 정기 업데이트
- 새 콘텐츠 추가
- 사이트맵 갱신
- 정기적인 크롤링 요청

## ⚠️ 주의사항

### DO ✅
- 정확한 메타 정보 입력
- 정기적인 사이트맵 업데이트
- 유효한 robots.txt 유지
- 모바일 최적화

### DON'T ❌
- 키워드 스팸
- 중복 콘텐츠
- 숨겨진 텍스트
- 자동 생성 콘텐츠
